XMPP install in UBUNTU

Step1=> apachefriends.org မှာ xampp for linux ကို download ဆွဲပါ။

Step2=> Terminal [ctrl+Alt+T] ဖွင့်ပါ။

Step3=> whoami လို့ရိုက်ပါ။ user name ကျလာပါလိမ့်မည်။

Step4=> cd /home/[user name]/Downloads => user name က အပေါ်ကကျလာတဲ့ username ကိုထည့်ပေးပါ။

Step5=> sudo chmod 755 [package name] => packagename မှာ download ဆွဲထားသော file name ဖြစ်သည်။ “xampp” လို့အစစလုံးရိုက်ပြီး Tab Key နှိပ်နိုင်သည်။

Step6=> ls -l [package name]

Step7 => sudo ./[package name]

Step8 => Xampp setup file ပွင့်လာပါလိမ့်မည်။

Step9 => Next=>Next->Next ဖြင့် Install တင်ပါ။

Step10 => Finish ပြီးသွားလျှင် Terminal [ctrl+Alt+T] ဖွင့်ပါ။

Step11 => sudo /opt/lampp/lampp start

Step12 => Apache, MySql, ProFTPD တို့ OK ဖြစ်နေလျှင် Apache Xampp Install တင်ခြင်းပြီးမြောက်ပါသည်။

Step13 => အကယ်၍ MySql error တတ်နေလျှင်

Step14 => Terminal [ctrl+Alt+T] တွင်

Step15 => sudo apt install net-tools ဟုရိုက်၍ Enter ခေါက်ပါ။

Step16 => အကယ်၍ net-tools ကို မသိပါက...

Step17 => sudo apt update လုပ်ပါ။

Step18 => net-tools ကို install ပြန်တင်ပါ။

Step19 => ပြီးလျှင် xampp start ပြန်လုပ်ပါ။

Step20 => sudo /opt/lampp/lampp start

Ok မှာစိုပြေနေပါလိမ့်မည်။


htdocs Folder ကို Permission ပေးခြင်း

Step21 => sudo chmod -R 777 /opt/lampp/htdocs
