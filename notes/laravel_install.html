Laravel Installation in Ubuntu

Step1=> sudo apt update && sudo apt upgrade

Composer Install

Step2=> sudo apt install curl

Step3=> sudo curl -s https://getcomposer.org/installer | /opt/lampp/bin/php

Step4=> sudo ln -s /opt/lampp/bin/php /usr/local/bin/php

Step5=> sudo mv composer.phar /usr/local/bin/composer

Step6=> composer --version

Node.js Install

Step7 => sudo apt install nodejs

Step8 => nodejs --version

npm Install

Step9 => sudo apt install npm

Step10 => npm --version

To Change Default Version

Step11 => command -v npm

Step12 => sudo npm install -g npm

Step13 => command -v npm

Step14 => /usr/local/bin/npm -v

Step15 => type npm

Step16 => hash -d npm

Step17 => npm -v

Laravel Project Download

Step18 => composer create-project --prefer-dist laravel/laravel blog
